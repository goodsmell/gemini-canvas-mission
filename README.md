# gemini-canvas-mission

Gemini Canvas 미션 결과물을 관리하는 저장소입니다.

## 목적

- Gemini Canvas로 제작한 웹앱 결과물을 기록하고 공유합니다.
- 앱별 요구사항, 배포 링크, 회고를 일관된 형식으로 남깁니다.

## 앱 소개

### 1. 유틸리티 앱 : 런닝구 (RunningGu)

러닝 기록과 신체 변화를 한눈에 파악할 수 있는 개인 맞춤형 건강 관리 앱입니다.

✨ 주요 기능

- 러닝 거리·칼로리·페이스 트래킹
- 기간별 활동 리포트
- 신체 변화 기록
- AI 기반의 신체 변화 분석 및 조언

### 2. 게임 : 페이커되기 (BECOME FAKER)

전설적인 프로게이머 Faker의 반응속도에 도전하는 30초 반응속도 챌린지 게임입니다.

✨ 주요 기능

- 실시간 반응속도 측정
- 구간별 등급 시스템(Faker부터 Bronze까지)

### 3. 학습 앱 : AI 기술 면접 마스터 (AI Interview Simulator)

신입 개발자를 대상으로 한 AI 기반 맞춤형 기술 면접 연습 플랫폼입니다.

✨ 주요 기능

- 실시간 AI 질문 생성
- 시간 제한 모드
- 답변에 대한 상세한 피드백 리포트

### 4. 페어 프롬프트 릴레이 앱 : 우아한 성장 트래커 (Elegant Growth Tracker)

우아한테크코스 미션과 학습 여정을 체계적으로 관리하고 시각화하는 도구입니다.

✨ 주요 기능

- 미션별 단계 관리
- 학습 로그 기록
- 차트를 통한 성장 곡선 시각화 및 결과 리포트 출력.
